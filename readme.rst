Git 技巧
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

* git clean -df   删除未跟踪文件，但不包括.gitignore中的
* git clean -n    测试删除的文件，实际不删除
* git clean -dfx  删除所有文件
